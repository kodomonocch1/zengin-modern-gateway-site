How to get an evaluation key

Phase 1 evaluation uses a provider-issued hosted API key in the Authorization header. This keeps the upstream contract unchanged while allowing low-friction evaluation.

  1. Subscribe on RapidAPI
  2. Request an evaluation key
  3. Test GET /v1/meta/health
  4. Test POST /v1/zengin/validate
  5. Move to Growth / Scale for production use
